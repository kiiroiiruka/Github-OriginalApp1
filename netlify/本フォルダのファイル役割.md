netlify.toml
→Netlify のビルド・Functions・リダイレクト設定（API 用）
→SPA 用リダイレクトは public/_redirects（本番デプロイ時に dist へコピー）

functions/
→サーバー側で動作する Netlify Functions

functions/send-verification-email.js
→POST /api/send-verification-email。登録時の確認メールをカスタム文面で送信

functions/send-email.js
→POST /api/send-email。ログイン済みユーザーが指定した宛先にメールを送信

functions/utils/common/
→どの API でも使う共通処理（CORS・レスポンス・JSON 解析など）

functions/utils/auth/
→Firebase 認証まわり（Admin SDK・トークン検証）

functions/utils/email/
→メール送信まわり（SMTP・確認メールの文面）
